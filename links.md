# About

The following is a list of third party utilities.

# ASCII

- [asciiflow](https://asciiflow.com): draw ASCII diagrams ([github](https://github.com/lewish/asciiflow))

# Google Chat

- [build card](https://addons.gsuite.google.com/uikit/builder): UI to build JSON payload for google chat cards
- [app examples](https://github.com/googleworkspace/google-chat-samples/tree/main/python): examples of google chat apps
- [send message via github action](https://github.com/marketplace/actions/google-chat-notification-action): github action that allows to send customizable messages to google chat via a space webhook url
- [known icons](https://developers.google.com/workspace/chat/format-messages#builtinicons): list with builtin icons to be passed in the `knowIcon` attribute.
